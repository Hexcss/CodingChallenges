# Challenge Description: FizzBuzz

## Challenge

Write a program that prints the numbers from 1 to 100. For multiples of 3, print "Fizz" instead of the number, and for the multiples of 5, print "Buzz". For numbers which are multiples of both 3 and 5, print "FizzBuzz".

## Problem Statement

You need to write a program that will print the numbers from 1 to 100, following the rules stated above. Your program should produce the output to the console.

Your program should print "Fizz" instead of the number for multiples of 3, "Buzz" instead of the number for multiples of 5, and "FizzBuzz" instead of the number for multiples of both 3 and 5.

+ The program should output to the console.
+ The program should output the numbers from 1 to 100.
+ For multiples of 3, print "Fizz" instead of the number.
+ For multiples of 5, print "Buzz" instead of the number.
+ For multiples of both 3 and 5, print "FizzBuzz" instead of the number.
